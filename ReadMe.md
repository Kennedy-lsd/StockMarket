## Getting Started

### Prerequisites

- Go version 1.18 or higher
- PostgreSQL
- [Make](https://www.gnu.org/software/make/) (optional for using Makefile)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Kennedy-lsd/StockMarket.git
   cd stockmarket
   ```
2. Simply write to the console:

   `make`
